# Professional Roadmap

## Phase 1: Current package
Native JUCE plugin/app, amp-style nonlinear DSP, cab coloration, gate, tone stack, UI, factory preset data, release scripts.

## Phase 2: Real cabinet IR loader
Add WAV/AIFF IR loading, minimum-phase conversion, IR browser, mic slots, dual cabs, latency compensation.

## Phase 3: Higher fidelity amp models
Replace generic waveshapers with per-model preamp stages, cathode follower approximation, dynamic power amp sag, transformer resonance, and negative feedback controls.

## Phase 4: Commercial UX
Preset browser, A/B compare, tuner, MIDI learn, undo/redo, scalable vector UI, signed installer, crash reporting.

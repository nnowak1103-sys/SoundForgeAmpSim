# Milestone 1

Goal: prove the repository builds a native audio application and plugin through GitHub Actions.

Included features:

- Valid JUCE/CMake layout
- GitHub Actions workflow
- VST3 + standalone target
- Parameterized DSP
- Basic professional architecture that can be expanded

Not yet included:

- Commercial-quality amp models
- Real cabinet IR library
- Photorealistic UI
- Code signing
- Installer

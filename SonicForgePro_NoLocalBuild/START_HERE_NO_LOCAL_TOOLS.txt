START HERE - NO LOCAL TOOLS BUILD

Do not run scripts\build_windows_release_LOCAL_REQUIRES_CMAKE.bat unless you have CMake and Visual Studio installed.

To build without installing anything on your PC:

1. Create a GitHub repository.
2. Upload this entire SonicForgePro_NoLocalBuild folder to the repository.
3. Go to the repository's Actions tab.
4. Click Windows Release Build.
5. Click Run workflow.
6. Download the artifact named SonicForgeProAmpSim-Windows.

That downloaded artifact is the Windows distribution output.

The file scripts\build_windows_release.bat in this package only prints these instructions and will not try to run CMake.
